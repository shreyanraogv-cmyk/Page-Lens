🔍 PageLens

PageLens is a simple webpage content analyser built with Streamlit. Paste in any URL and instantly pull out structured content from the page — no technical knowledge needed.


✨ Features


Headings — extracts all H1, H2, and H3 tags from a page
Paragraphs — pulls all readable paragraph text
Links — lists every hyperlink along with its display text and destination URL
Title & Meta — grabs the page title and meta description


Results are displayed as a clean table and can be downloaded as a CSV file.
